# Meili Android SDK — public distribution

Public distribution channel for the **Meili Android SDK** (`meili.travel:ux-native-android-sdk`).

The compiled `.aar` is served from this repository as a Maven repository over **GitHub Pages**, so **no credentials are required** to consume it. The SDK source lives in a private repository and is not part of this distribution.

## Installation

Add the Meili Maven repository in your `settings.gradle.kts`:

```kotlin
dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
        maven { url = uri("https://meili-travel-tech.github.io/ux-native-android/") }
    }
}
```

Add the dependency in your app module's `build.gradle.kts`:

```kotlin
dependencies {
    implementation("meili.travel:ux-native-android-sdk:VERSION")
}
```

Replace `VERSION` with the latest from [Releases](https://github.com/meili-travel-tech/ux-native-android/releases).

## Requirements

- `minSdkVersion` 27 or higher
- Jetpack Compose

## How this repository is maintained

This repository is generated by CI. The `docs/` directory holds the Maven repository and is updated automatically on each SDK release by the publishing pipeline in the private source repository. Do not edit `docs/` by hand.

The `docs/.nojekyll` marker disables Jekyll so GitHub Pages serves the artifacts as static files.
